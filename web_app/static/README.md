# 静态资源说明

## 目录结构

```
web_app/static/
├── css/              # 样式文件
│   └── main.css      # 主样式文件
├── images/           # 图片资源
│   ├── background/   # 背景图片
│   ├── map/          # 地图相关图片
│   ├── spots/        # 景点图片
│   └── food/         # 美食图片
└── icons/            # 图标资源
```

## 图片格式和大小建议

- 背景图片：建议使用JPG格式，分辨率1920x1080或更高
- 景点和美食图片：建议使用JPG格式，分辨率1200x800或更高
- 图标资源：建议使用PNG格式，大小48x48像素

## 如何使用

1. 将图片资源按照上述目录结构放置
2. 系统会自动引用这些资源
3. 当图片不存在时，会使用默认图片（如果提供）

## 注意事项

- 请确保图片文件名与代码中引用的名称一致
- 图片大小不宜过大，建议压缩后使用，以提高加载速度
- 图标资源建议使用透明背景的PNG格式
